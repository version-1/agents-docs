# Remove Branches

## 目的

main、develop 以外のローカルブランチを整理する。

## 絶対ルール

- main と develop は削除しない。
- カレントブランチは削除しない。
- リモートブランチは削除しない。
- コード変更や未保存の作業ツリー変更は行わない。
- 未マージブランチを強制削除する前にユーザーへ確認する。

## 手順

1. 現在のブランチと作業ツリー状態を確認する。
2. 削除候補のローカルブランチを列挙する。
3. main、develop、カレントブランチを候補から外す。
4. 未マージかどうかを確認し、強制削除が必要な場合は確認する。
5. 削除したブランチと残したブランチを報告する。
