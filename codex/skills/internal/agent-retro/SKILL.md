# Agent Retro

直近のタスクから「最初に知っていれば遠回りしなかった」知見を抽出し、次のいずれかへ反映する。

- skill
- CLAUDE.md
- AGENTS.md

## 手順

1. 実際に詰まった点、誤解した点、再利用できる判断基準を分ける。
2. 反映先を選ぶ。特定タスクの手順は skill、全体ルールは AGENTS.md / CLAUDE.md を優先する。
3. 追加する文を短く具体的にする。
4. 反映前に、反映先と変更内容をユーザーへ確認する。
