# Code Go

## 目的

Go の実装やテストで、既存コードに自然に馴染む判断をするための参照 skill。

## 使い方

- まず [../code-general/SKILL.md](../code-general/SKILL.md) を読む。
- Go 実装では references/lang.md を必ず読む。
  - このファイルは、この SKILL.md と同じディレクトリの references/ 配下にある。
  - 見つからない場合は ~/.claude/skills/code-go/references/lang.md を試す。
  - それでも見つからない場合はユーザーに報告し、references なしで進めない。
- Go のテストを追加、修正、レビューする場合は references/test.md も必ず読む。
  - このファイルは、この SKILL.md と同じディレクトリの references/ 配下にある。
  - 見つからない場合は ~/.claude/skills/code-go/references/test.md を試す。
  - それでも見つからない場合はユーザーに報告し、references なしで進めない。
